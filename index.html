ABOUT

learn music theory and practice touch typing with the dynamic virtual piano keyboardle.

it makes use of the fact that music is composed in key signatures,
this means that only certain notes will be played for any given song at a time.


thanks for playing.

DEVELOPMENT

keyboardle was made entirely in HTML, CSS and vanilla JS as my winter break project, no external libraries were used.

approximately 1000 lines of code were written in the process.

request songs and send feedback at j.pruessing28@gmail.com.
